    Class FluidProperties

    Fluid properties for shapes that act as fluid regions.

    Controls density, viscosity, and per-fluid gravity override. Internally wraps a ZPP_FluidProperties and is registered as the public nape.phys.FluidProperties class in the compiled namespace.
