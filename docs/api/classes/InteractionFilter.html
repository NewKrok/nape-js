    Class InteractionFilter

    Bit-mask based interaction filter for controlling which shapes collide, sense, or interact as fluids.

    Internally wraps a ZPP_InteractionFilter and is registered as the public nape.dynamics.InteractionFilter class in the compiled namespace.
