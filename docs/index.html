API at a Glance

Core

  • Space — physics world
  • Body — rigid body
  • Vec2 — 2D vector
  • AABB — bounding box

Shapes

  • Circle — radius-based
  • Polygon — convex hull
  • Polygon.box(w, h)
  • Polygon.regular(r, r, n)

Constraints

  • PivotJoint
  • DistanceJoint
  • WeldJoint
  • AngleJoint
  • MotorJoint
  • LineJoint
  • PulleyJoint

Callbacks

  • InteractionListener
  • BodyListener
  • ConstraintListener
  • PreListener
  • CbType + CbEvent
